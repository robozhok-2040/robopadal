# RoboPadal Americano

Mobile-first web app for running an 8-player Americano padel session (2 courts, 7 rounds).

## Tech
- Vite + React + TypeScript
- Simple CSS (no heavy UI library)
- Session persistence in `localStorage`

## Local run
```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.

## Build
```bash
npm run build
npm run preview
```
